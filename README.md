# 多Agent狼人杀模拟沙盒

使用 Python 和 LangGraph 构建的智能狼人杀游戏系统，支持 AI 自动对战和玩家参与两种模式。

## 功能特性

- 多 Agent 系统：每个玩家都有独立的性格、记忆和推理能力
- 性格生成：根据简短描述自动生成详细的角色性格提示词（如"鲁莽的张飞"）
- 两种游戏模式：
  - **沙盒模式**：观看 AI 们对战，可以看到所有信息
  - **参与模式**：你也是玩家之一，与 AI 一起游戏
- 完整的狼人杀规则实现
- 记忆管理系统：每个 Agent 都有自己的记忆
- 智能推理引擎：Agent 能够根据游戏状态进行推理和决策

## 项目结构

```
ai-werewolf-game/
├── core/              # 核心模块
│   ├── roles.py       # 角色定义
│   └── game_state.py  # 游戏状态管理
├── agents/            # Agent 系统
│   ├── personality_generator.py  # 性格生成器
│   ├── memory_manager.py         # 记忆管理器
│   └── reasoning_engine.py       # 推理引擎
├── game/              # 游戏流程
│   └── graph.py       # LangGraph 游戏流程图
├── main.py            # 主程序入口
├── requirements.txt   # 依赖包
└── .env.example       # 环境变量示例
```

## 安装

1. 克隆或下载项目
2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 配置环境变量：
```bash
cp .env.example .env
# 编辑 .env 文件，填入你的 OpenAI API Key
```

## 使用方法

运行游戏：
```bash
python main.py
```

按照提示选择游戏模式并开始游戏！

## 角色列表

### 村民阵营
- 预言家、女巫、猎人、白痴、守卫、丘比特、长老、驯熊人、守墓人

### 狼人阵营
- 狼人、白狼王、狼美人、狼先知、种狼

### 第三方阵营
- 情侣（当一人是好人一人是狼人时）

## 配置说明

在 `main.py` 中可以修改默认的玩家配置，自定义每个玩家的名字和性格描述。

## 注意事项

- 需要有效的 OpenAI API Key
- 游戏可能需要较长时间运行，因为每个决策都需要调用 LLM
- 建议使用 GPT-3.5-turbo 或更高版本的模型

## 游戏规则

详细的狼人杀游戏规则已内置在系统中，包括：
- 夜晚阶段（狼人杀人、预言家查验、女巫用药、守卫守护等）
- 白天阶段（公布死亡信息、发言讨论、投票驱逐）
- 特殊角色技能
- 胜负判定
